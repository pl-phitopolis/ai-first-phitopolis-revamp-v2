# Enhancements - 2026-02-23

This document outlines the planned action points for fine-tuning the Phitopolis website.

## 1. Landing Page Loading Animation
- **Goal**: Provide a visually engaging experience while assets (video, images, etc.) are loading.
- **Action Points**:
    - Implement an initial loader that appears before the landing page content is fully ready.
    - Animation should feature an animated Phitopolis logo.
    - The design of the loading animation must fit the overall theme and aesthetic of the site.

## 2. Careers Page
- **Goal**: Improve usability and navigation for job seekers.
- **Action Points**:
    - **Filtering**: Implement a flexible filtering system that is ready for various types beyond just Employment Type (e.g., Department, Location, Seniority Level). It should be architected to easily accommodate new filter categories.
    - **Pagination**: Add pagination to manage large lists of career opportunities effectively.

## 3. Blog Page
- **Goal**: Enhance content discoverability and browsing experience.
- **Action Points**:
    - **Filter by Author**: Enable users to view blog posts specifically written by selected authors.
    - **Pagination**: Implement pagination for the blog listing to improve page performance and navigation.
